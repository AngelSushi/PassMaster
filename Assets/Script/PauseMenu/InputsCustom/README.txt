--- Système d'input custom avec UI compatible nouveau Input System Unity ---

/!\ Si vous avez une action avec plusieurs inputs il faut marquer le nom de l'action x fois le nombre d'input dans la variable actionsName

/!\ Si vous avez une action avec qu'un seul input actionIndex = 0
	Sinon actionIndex = 1 et va jusqu'à votre nombre d'input dans votre action

Il faut bien mettre les actionsIndex en fonction de l'ordre des noms dans actionName
